# Media Manager

This is the main source code repository for my picture manager.

## Installation

1. Clone the repo
   ```bash
   git clone https://github.com/ericj974/media_manager.git
   ```
2. Setup the conda environment
   ```bash
   conda install -f environment_3.yml
   ```
3. Run the main app
   ```bash
   conda activate media
   python main_app.py
